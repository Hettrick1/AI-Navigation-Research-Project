# AI-Navigation-Research-Project

The goal of this research project is to dive into AI Navigation in video games. With this project I will understand what solutions exists to make an AI moves, which algorithms to use, which data structure to chose.
Then I'm gonna implement, in my own game engine, a way to make an agent move freely in a scene. For that I'm gonna use the physics to generate a 2.5D grid. 
Using A* I will determine, for my agent, the shortest path from it's original location to a certain target. I will then dive into some path smoothing algorithms to make the movements looks the most natural as possible.

For this Reseach project I will use my own game engine : [Zephyrus](https://github.com/Hettrick1/Zephyrus-Engine). This Repo is a fork of the 1st of April 2026 of the Zephyrus-Engine Repository.

I will not use any AI navigation library because the interest of this project is to do it myself. 
In this research project I made a big use of external libraries, especially for all the physics computation. This external library is [Bullet3](https://github.com/bulletphysics/bullet3).
