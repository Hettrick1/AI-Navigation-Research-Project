## Third-Party Software Notices

Zephyrus is built using the following libraries and tools:

- [**Premake5**](https://premake.github.io/) (BSD-3) - Build configuration tool
- [**Bullet Physics 3.25**](https://github.com/bulletphysics/bullet3) (Zlib) - 3D Collision & Physics engine
- [**FreeType 2.6.1**](https://freetype.org/) (FTL/GPLv2) - Font rendering
- [**GLEW 2.2.0**](http://glew.sourceforge.net/) - OpenGL Extension Wrangler
- [**GLFW 3.4**](https://www.glfw.org/) (Zlib) - Windowing and input handling
- [**Dear ImGui 1.92.2b**](https://github.com/ocornut/imgui) (MIT) - Immediate Mode GUI for tools
- [**ImGuizmo 1.83**](https://github.com/CedricGuillemet/ImGuizmo) (MIT) - Gizmo widgets for the scene editor
- [**RapidJSON 1.1.0**](https://github.com/Tencent/rapidjson) (MIT) - Fast JSON parser/generator
- [**stb_image**](https://github.com/nothings/stb) (Public Domain/MIT) - Image loading utilities
- [**stduuid 1.2.3**](https://github.com/mariusbancila/stduuid) (MIT) - C++17 cross-platform UUID generator
- [**TinyObjLoader**](https://github.com/tinyobjloader/tinyobjloader) (MIT) - Tiny but powerful .obj loader
